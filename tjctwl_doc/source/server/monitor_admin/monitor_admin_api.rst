设备中心 API
^^^^^^^^^^^^

- 导出设备列表

导出设备列表excel表格

::

  download_device_list
  
  参数：
     联网单位
     设备编码
     设备名称
     状态
     设备型号
     信号值
     具体位置
     电压
     地址：建华里2号楼1门3楼左侧
   返回值：
      excel表格
